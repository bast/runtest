runtest
=======

Numerically tolerant end-to-end test library for scientific codes.

Tested with Python 2.7 and 3.6.

.. figure:: https://github.com/bast/runtest/raw/master/img/xanathar.jpg
   :alt: Xanathar


Installation
------------

::

  $ pip install runtest
