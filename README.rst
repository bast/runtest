runtest
=======

Numerically tolerant end-to-end test library for research software.

Tested with Python 3.6, 3.7, 3.8.

.. figure:: https://github.com/bast/runtest/raw/master/img/xanathar.jpg
   :alt: Xanathar


Installation
------------

::

  $ pip install runtest
